# __mftk__

A UI toolkit inspired by 1960s mainframe control panels, written in C

## Dependencies: 
  + SDL3
  + SDL3_image
  + cJSON

## Make targets
  + __build__
    + builds the library
    + default target
  + __test__
    + builds the test program
  + __test_run__
    + builds and runs the test program
  + __clean__
    + removes intermediate object files
  + __clean_all__
    + removes all object files
